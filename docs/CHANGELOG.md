# Changelog — 知天

## v1.3 (2026-05-15) — 文化深度 + 体验打磨

### 新功能
- **诗词收藏夹**：首页诗词卡片支持 ❤️ 收藏，设置页"我的诗集"查看/管理收藏
- **今日宜忌**：基于天气/温度/风速/季节的生活建议卡片（宜/忌双栏）
- **诗词左右滑动**：HorizontalPager 5 页探索更多诗词 + 圆点指示器
- **首页问候语**：时段个性化（早安/上午好/午间/下午好/傍晚/晚上/夜深）
- **启动页诗词**：骨架屏加载时居中显示大字诗词（每小时轮换）

### 内容扩展
- **诗词库扩展至 1200 首**（从 250 首增长 4.8x）
- 迁移到 JSON assets 架构（10 个分类文件）
- 支持 3 年以上不重复展示
- 来源：唐诗三百首、宋词三百首、千家诗、古诗十九首等公共领域作品

### 视觉提升
- **卡片入场动画**：staggered slideIn + fadeIn（60ms 间隔）
- **温度视觉优化**：72sp 极细字重 + 垂直渐变色填充
- **日历今日脉冲**：今天日期格子 scale 呼吸动画 + 自动选中

### 性能优化
- 诗词 JSON 加载移到后台线程（Dispatchers.IO），不阻塞主线程
- 粒子数优化：雷暴 120→60，雨 80→50，雪 80→45
- 分享图片从 6MB 缩小到 1.4MB（720×960 RGB_565）
- AQI API 首次 403 后不再重试（节省免费版配额）
- 低内存设备（isLowRamDevice）永久禁用动画

### 安全加固
- FileProvider 路径从 `/` 收窄到 `share/`
- BootCompletedReceiver 添加权限保护
- Room 添加 fallbackToDestructiveMigrationOnDowngrade

### 稳定性修复
- DailyPoetry 线程安全（@Synchronized + @Volatile）
- HomeViewModel.freshDataLoaded 改用 AtomicBoolean（消除竞态）
- 诗词池初始化为 fallback（加载完成前不为 null）

## v1.2 (2026-05-15) — 智能预警 + 动画体系

### 新功能
- **日程天气预警引擎**（AlertEngine）
  - 15 天预报 × 日程交叉匹配
  - 室内事件智能过滤（20+ 关键词）
  - 降水量阈值（>10mm）
  - 小时级精确匹配（±1 小时）
  - 灾害预警 × 日程关联
  - 本地推送通知（未来 3 天）
- **诗词桌面 Widget**（Jetpack Glance）
- **天气图标微动画**（64dp Canvas）
- **背景转场动画**（700ms ease-in-out）
- **动画降级管理**（低电量/省电模式自动禁用）
- **深色/浅色模式**影响渐变背景

### 改进
- 气象灾害预警提升到首页最醒目位置
- 预警卡片使用颜色边框（红/橙/黄/蓝）
- 分享支持微信/微博等社交 App
