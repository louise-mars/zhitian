# 天气日历 — UI 设计规范 v2

## 一、设计原则

1. **信息分层**：第一屏给决策信息，详细数据按需展开
2. **极简默认**：湿度/气压/UV 等折叠隐藏，不堆砌数据
3. **天气+日历融合**：每一天 = 天气 + 事件，不是上下拼接
4. **Minimal / Soft / Airy / Context-aware**：极简、柔和、大留白、随天气变化

---

## 二、信息架构

```
首页（核心，80%时间在这里）
├── 顶部栏（城市选择 + 日期 + 农历）
├── 当前天气卡（Hero Card）
├── 小时预报（横滑）
├── 天气+日历融合卡（仅3天）
└── 详细指标（折叠）

日历页
├── 月视图（带天气图标 + 农历）
└── 日期详情（BottomSheet：天气+事件）

设置页（极简）
├── 城市管理
├── 温度单位（°C / °F）
└── 主题偏好
```

只做 3 个 Tab 以内，保持轻量。

---

## 三、首页

### 整体布局：单页纵向流 + 全页天气动态背景

页面背景根据当前天气状态显示对应渐变色，所有卡片使用毛玻璃效果浮于其上。

```
┌──────────────────────────────┐  ← 全页天气渐变背景
│                              │
│  [北京 ▼]            [📅][⚙️]│  ← 顶部栏
│  4月16日 星期三 · 三月十九    │
│                              │
│  ┌────────────────────────┐  │
│  │        ☀️              │  │
│  │       23°              │  │  ← Hero Card（毛玻璃）
│  │   体感 21° · 适合外出   │  │     屏幕 40-50%
│  └────────────────────────┘  │
│                              │
│  → 08:00 ☀️ 12° → 09:00 🌤 13° │  ← 小时预报横滑
│                              │
│  ┌────────────────────────┐  │
│  │ ◉ 今天 4/16 ☀️ 8°~15° │  │  ← 融合卡（仅3天）
│  │   10:00 Team Meeting   │  │     今天默认展开
│  │   18:00 Gym            │  │     手风琴交互
│  │ ○ 明天 4/17 🌧 6°~12°  │  │
│  │ ○ 后天 4/18 ⛅ 9°~16°  │  │
│  └────────────────────────┘  │
│                              │
│  ┌────────────────────────┐  │
│  │ ▸ 详细信息              │  │  ← 折叠区
│  └────────────────────────┘  │
│                              │
└──────────────────────────────┘
```

### 3.1 顶部栏（Header）

```
[北京 ▼]                [📅] [⚙️]
4月16日 星期三 · 三月十九
```

- **左侧**：城市名 + 下拉箭头（点击弹出城市选择器 BottomSheet）
- **右侧**：日历图标（进入月视图）+ 设置图标
- **第二行**：公历日期 + 星期 + 农历日期
- 农历节日/节气时，农历文字变为红色/特殊色
- 文字颜色：白色或深色，根据天气背景自动适配

### 3.2 当前天气卡（Hero Card）

```
┌──────────────────────────┐
│           ☀️             │
│          23°             │
│    体感 21° · 适合外出    │
└──────────────────────────┘
```

- **占比**：屏幕高度 40-50%，确保下方融合卡头部露出
- **背景**：半透明白底（15% 透明度）+ 模糊（毛玻璃）
- **内容层级**（从上到下）：
  1. 天气图标（Material 风格 emoji，48sp）
  2. 温度（视觉中心，**40-48sp**，FontWeight.Thin）
  3. 体感温度 + 可行动提示（bodyMedium，70% 透明度）
- **可行动提示示例**：
  - "适合外出"
  - "风大，带件外套"
  - "2小时后有雨"
- 圆角：24dp
- 极轻阴影（不用 Material 默认重阴影）

### 3.3 小时预报（横向滚动）

```
→ 08:00 ☀️ 12° → 09:00 ☀️ 13° → 10:00 ☁️ 14° → 11:00 🌧 13°
```

- 横滑，卡片宽度统一
- 每个 chip 只显示：**时间 + 天气图标 + 温度**
- 当前小时高亮（背景 25% 白色透明）
- 其他 chip：10% 白色透明
- **不放**风速、湿度
- 圆角：16dp
- 横滑惯性要顺畅

### 3.4 天气+日历融合卡（核心创新，仅 3 天）

只展示**今天、明天、后天**，更多天数去日历页查看。

#### 元素优先级

| 元素 | 优先级 | 始终可见 |
|------|--------|---------|
| 天气图标 | 高 | ✅ |
| 温度范围 | 高 | ✅ |
| 事件数量 | 中 | ✅（有事件时）|
| 事件详情 | 低 | 点击后展开 |

#### 交互规则

- **手风琴模式**：同一时间只展开一个
- **今天默认展开**
- **无事件的日子**：只显示天气行，无展开箭头
- **有事件的日子**：右侧 "🗓 N" + 展开箭头

#### 收起状态

```
今天  4/16  ☀️  8° ~ 15°        🗓 2  ▸
```

#### 展开状态

```
今天  4/16  ☀️  8° ~ 15°        🗓 2  ▾
────────────────────────────────
  ● 10:00  Team Meeting
  ● 18:00  Gym
```

#### 视觉

- 卡片背景：毛玻璃（半透明白底）
- 今天行：字体加粗
- 事件行：左侧彩色圆点（取系统日历颜色）+ 时间 + 标题
- 全天事件显示"全天"替代时间
- 圆角：16dp

### 3.5 可折叠详情区

```
▸ 详细信息                    ← 默认收起

▾ 详细信息                    ← 展开后
  湿度        风速        UV指数
  65%       12 km/h     中等
```

- 默认收起，只显示标题
- 展开后三列等分
- 毛玻璃卡片，圆角 16dp

### 3.6 首页手势

| 手势 | 行为 |
|------|------|
| 左右滑 | 切换日期焦点（今天 ↔ 明天 ↔ 后天），天气卡内容+渐变色跟随切换 |
| 下拉 | 刷新天气数据 |
| 点击融合卡 | 展开/收起事件（手风琴） |
| 点击城市名 | 弹出城市选择器 |

- 左右滑时底部加页面指示器（3个小圆点）
- 滑动切换时融合卡列表中对应日期自动展开

## 四、日历月视图

### 整体结构

```
┌──────────────────────────────┐  ← 天气动态背景（降低饱和度）
│  ←  日历          2026年4月   │
├──────────────────────────────┤
│  一   二   三   四   五   六   日 │
│             1    2    3    4    5 │
│            初三  初四  初五 清明 初七│  ← 农历小字
│  6    7    8    9   10   11   12 │
│  ☀️       🌧   ⛅                │  ← 有预报的日期带天气图标
│  ●              ●               │  ← 有事件的日期带圆点
│  ...                            │
└──────────────────────────────┘

点击某天 → 底部弹出 BottomSheet：
┌──────────────────────────────┐
│  ── 拖拽条 ──                 │
│  4月16日 星期三                │
│  农历 三月十九                 │
│                              │
│  ☀️ 晴  8° ~ 15°             │
│  适合外出                     │
│  ────────────────────        │
│  ● 10:00  Team Meeting       │
│  ● 18:00  Gym                │
│                              │
│  暂无更多日程                  │
└──────────────────────────────┘
```

### 4.1 月份导航

- 左右箭头切换月份
- 标题格式："2026年4月"

### 4.2 日期格子

- 每格内容：日期数字 + 农历小字 + 天气图标（有预报时）+ 事件圆点
- **不放温度**，保持干净
- 今天：圆形背景高亮（secondaryContainer）
- 选中日期：primaryContainer 背景
- 农历节日/节气：红色/特殊色文字，替代普通农历日显示

### 4.3 日期详情（BottomSheet）

- 点击日期后从底部弹出，**不跳新页面**
- 包含：公历 + 农历 + 天气 + 可行动提示 + 事件列表
- 无事件显示"暂无日程"
- BottomSheet 顶部圆角 28dp

### 4.4 配色

- **跟随天气动态背景**，但饱和度降低 30-40%
- 确保日历网格文字可读性
- 日期格子背景用更高透明度的毛玻璃

## 五、城市选择器（BottomSheet）

点击首页顶部城市名触发，从底部弹出半屏。

```
┌──────────────────────────────┐
│  ── 拖拽条 ──                 │
│  选择城市                     │
│  ┌────────────────────────┐  │
│  │ 🔍 搜索城市...          │  │
│  └────────────────────────┘  │
│                              │
│  📍 当前定位                  │
│     北京  22°  ☀️            │
│                              │
│  ★ 收藏城市                   │
│     上海  19°  🌤️            │
│     深圳  26°  ⛅             │
│     东京  15°  🌧️            │
│                              │
│  ＋ 添加城市                  │
└──────────────────────────────┘
```

### 规则

- 当前 GPS 定位城市始终在顶部，带定位图标
- 收藏城市：显示城市名 + 当前温度 + 天气图标
- 搜索：输入城市名，调用 geocoding API
- 选中后 BottomSheet 收起，首页刷新
- 长按收藏城市可删除（带确认对话框）

## 七、农历支持

### 显示位置

1. **首页顶部栏**：公历日期旁（如"· 三月十九"）
2. **日历月视图**：每个日期格子下方小字
3. **BottomSheet 日期详情**：完整农历（如"农历 三月十九"）

### 特殊日期标注优先级

1. **农历节日**（春节、中秋、端午、元宵等）→ 红色文字
2. **节气**（立春、清明、冬至等）→ 特殊色（如绿色）
3. **普通农历日**（初一、初二...）→ 默认淡灰色

优先级高的替代低的显示。例如清明节那天显示"清明"而非"三月初六"。

## 八、视觉规范

### 8.1 动态背景配色

全页渐变，根据当前天气实时变化：

| 天气状态 | 渐变起始色 | 渐变结束色 |
|---------|-----------|-----------|
| 晴天（白天）| #4FC3F7 | #0288D1 |
| 多云 | #B0BEC5 | #78909C |
| 雨天 | #546E7A | #37474F |
| 雪天 | #E1F5FE | #B3E5FC |
| 夜间 | #1A237E | #0D47A1 |
| 雷暴 | #424242 | #212121 |

日历页使用同色系但降低饱和度 30-40%。

### 8.2 毛玻璃卡片

| 组件 | 白色透明度 | 模糊 |
|------|-----------|------|
| Hero Card | 15% | 有 |
| 融合卡 | 12% | 有 |
| 小时 chip（普通）| 10% | 轻微 |
| 小时 chip（当前）| 25% | 轻微 |
| 详情折叠区 | 12% | 有 |

暗色模式下白色透明改为黑色透明。

阴影：极轻，不用 Material 默认重阴影。

### 8.3 字体层级

| 用途 | 大小 | 字重 | 透明度 |
|------|------|------|--------|
| 温度（焦点）| 40-48sp | Thin | 100% |
| 天气描述 | 18-22sp | Normal | 100% |
| 体感/提示 | 14sp | Normal | 70% |
| 融合卡日期 | 16sp | Normal / Bold(今天) | 100% |
| 融合卡温度 | 14sp | Normal | 100% |
| 事件时间 | 12sp | Normal | 60% |
| 农历日期 | 12sp | Normal | 60% |
| 详情数值 | 18sp | SemiBold | 100% |
| 详情标签 | 12sp | Normal | 60% |

### 8.4 圆角规范

| 组件 | 圆角 |
|------|------|
| Hero Card | 24dp |
| 融合卡 | 16dp |
| 小时 chip | 16dp |
| 详情折叠区 | 16dp |
| 日历日期格 | 8dp |
| BottomSheet | 顶部 28dp |
| 搜索框 | 12dp |

### 8.5 间距

| 位置 | 间距 |
|------|------|
| 页面水平内边距 | 20dp |
| 卡片之间 | 12dp |
| Hero Card 内边距 | 24dp |
| 融合卡内边距 | 16dp |
| 融合卡行间距 | 8dp |
| 小时 chip 间距 | 4dp |

### 8.6 暗色模式

- 跟随系统自动切换（设置页可手动覆盖）
- 天气渐变色保持不变（本身适配暗色场景）
- 毛玻璃底色：白色透明 → 黑色透明
- 文字颜色自动适配

## 九、动效规范

| 交互 | 动效 | 时长 |
|------|------|------|
| 左右滑切换日期 | 天气卡内容淡入淡出 + 渐变色过渡 | 300ms |
| 融合卡展开/收起 | 高度动画（animateContentSize）| 250ms |
| 下拉刷新 | Material 3 PullToRefresh | 系统默认 |
| BottomSheet 弹出 | 从底部滑入 | 250ms |
| 天气渐变变化 | 颜色渐变过渡 | 500ms |
| 页面切换（首页↔日历） | 共享元素过渡 | 300ms |

## 十、页面流转

```
首页 ──点击📅──→ 日历月视图
首页 ──点击城市名──→ 城市选择器（BottomSheet）
首页 ──点击⚙️──→ 设置页
首页 ──左右滑──→ 切换日期焦点（3天）
日历页 ──点击日期──→ 日期详情（BottomSheet）
日历页 ──← 返回──→ 首页
```
